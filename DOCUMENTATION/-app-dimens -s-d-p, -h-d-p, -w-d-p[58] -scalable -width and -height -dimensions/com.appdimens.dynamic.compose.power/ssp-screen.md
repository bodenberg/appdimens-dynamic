EN Extension for TextUnit (Sp) with dynamic scaling based on **Smallest Width (swDP)**. Uses the base value by default, but when the device matches [uiModeType](ssp-screen.md) AND the screen metric for [qualifierType](ssp-screen.md) is >= [qualifierValue](ssp-screen.md), it uses [screenValue](ssp-screen.md) instead. Usage example: `30.sspScreen(50, UiModeType.TELEVISION, DpQualifier.SMALL_WIDTH, 600)` → 30.pwssp by default, 50.pwssp on television with sw >= 600.
