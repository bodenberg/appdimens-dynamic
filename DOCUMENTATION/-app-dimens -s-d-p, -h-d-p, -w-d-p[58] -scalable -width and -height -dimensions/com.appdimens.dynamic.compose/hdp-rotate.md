# hdpRotate

[jvm]

EN Extension for Dp with dynamic scaling based on **Screen Height (hDP)**. Uses the base value by default, but when the device is in the specified [orientation](hdp-rotate.md), it uses [rotationValue](hdp-rotate.md) scaled with the given [finalQualifierResolver](hdp-rotate.md). Usage example: `30.hdpRot(45, DpQualifier.HEIGHT, Orientation.LANDSCAPE)` → 30.hdp by default, 45 scaled by HEIGHT in landscape.

PT Extensão para Dp com dimensionamento dinâmico baseado na **Altura da Tela (hDP)**. Usa o valor base por padrão, mas quando o dispositivo está na [orientation](hdp-rotate.md) especificada, usa [rotationValue](hdp-rotate.md) escalado com o [finalQualifierResolver](hdp-rotate.md) dado. Exemplo de uso: `30.hdpRot(45, DpQualifier.HEIGHT, Orientation.LANDSCAPE)` → 30.hdp por padrão, 45 escalado por HEIGHT no paisagem.
