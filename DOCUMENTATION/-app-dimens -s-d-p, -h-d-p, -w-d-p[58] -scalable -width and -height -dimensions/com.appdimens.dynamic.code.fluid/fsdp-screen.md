Uses the base value by default, but when the device matches [uiModeType](fsdp-screen.md) AND the screen metric for [qualifierType](fsdp-screen.md) is >= [qualifierValue](fsdp-screen.md), it uses [screenValue](fsdp-screen.md) instead.
