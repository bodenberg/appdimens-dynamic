EN Extension for Int with dynamic scaling based on **Screen Width (wDP)**. Uses the base value by default, but when the device matches [uiModeType](logwdp-screen.md) AND the screen metric for [qualifierType](logwdp-screen.md) is >= [qualifierValue](logwdp-screen.md), it uses [screenValue](logwdp-screen.md) instead.

PT Extensão para Int com dimensionamento dinâmico baseado na **Largura da Tela (wDP)**. Usa o valor base por padrão, mas quando o dispositivo corresponde ao [uiModeType](logwdp-screen.md) E a métrica de tela para [qualifierType](logwdp-screen.md) é >= [qualifierValue](logwdp-screen.md), usa [screenValue](logwdp-screen.md) no lugar.
