EN Extension for TextUnit (Sp) with dynamic scaling based on **Screen Width (wDP)**. Uses the base value by default, but when the screen metric for [qualifierType](wsp-qualifier.md) is >= [qualifierValue](wsp-qualifier.md), it uses [qualifiedValue](wsp-qualifier.md) instead. Usage example: `30.wspQualifier(50, DpQualifier.WIDTH, 600)` → 30.logwsp by default, 50.logwsp when screenWidthDp >= 600.

PT Extensão para TextUnit (Sp) com dimensionamento dinâmico baseado na **Largura da Tela (wDP)**.
