# sdpPh

fun [Number](https://kotlinlang.org/api/core/kotlin-stdlib/kotlin/-number/index.html).[sdpPh](sdp-ph.md)(context: [Context](https://developer.android.com/reference/kotlin/android/content/Context.html)): [Float](https://kotlinlang.org/api/core/kotlin-stdlib/kotlin/-float/index.html)

EN Extension for Int with dynamic scaling based on **Smallest Width (swDP)**, but in portrait orientation it acts as **Screen Height (hDP)**. Usage example: `32.sdpPh(context)`.

PT Extensão para Int com dimensionamento dinâmico baseado na **Smallest Width (swDP)**, mas na orientação retrato atua como **Altura da Tela (hDP)**. Exemplo de uso: `32.sdpPh(context)`.
