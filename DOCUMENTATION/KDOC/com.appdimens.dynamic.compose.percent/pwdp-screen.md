EN Extension for Dp with dynamic scaling based on **Screen Width (wDP)**. Uses the base value by default, but when the device matches [uiModeType](pwdp-screen.md) AND the screen metric for [qualifierType](pwdp-screen.md) is >= [qualifierValue](pwdp-screen.md), it uses [screenValue](pwdp-screen.md) instead. Usage example: `30.wdpScreen(50, UiModeType.TELEVISION, DpQualifier.WIDTH, 600)` → 30.wdp by default, 50.wdp on television with width >= 600.
