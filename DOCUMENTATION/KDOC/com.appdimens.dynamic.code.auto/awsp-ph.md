# awspPh

fun [Number](https://kotlinlang.org/api/core/kotlin-stdlib/kotlin/-number/index.html).[awspPh](awsp-ph.md)(context: [Context](https://developer.android.com/reference/kotlin/android/content/Context.html)): [Float](https://kotlinlang.org/api/core/kotlin-stdlib/kotlin/-float/index.html)

EN Extension for Int with dynamic scaling based on the **Screen Width (wDP)**, but in portrait orientation it acts as **Screen Height (hDP)**. Usage example: `100.awspPh(context)`.
