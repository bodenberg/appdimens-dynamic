**AppDimens** é uma biblioteca projetada para facilitar a criação de layouts Android mais consistentes, adaptáveis e escaláveis, oferecendo utilidades modernas para trabalhar com dimensões em diferentes dispositivos e densidades de tela.

---

## 📢 Próxima versão

## ⚠️ Versão **3.0.0** em desenvolvimento

A próxima grande versão da biblioteca (**3.0.0**) está atualmente **em desenvolvimento ativo** e trará uma série de **novidades importantes, melhorias arquiteturais e otimizações de performance**.

Entre os objetivos desta nova versão estão:

✨ **Melhor desempenho geral**
✨ **APIs mais modernas e consistentes**
✨ **Melhor suporte para Compose e multiplataforma**
✨ **Redução de overhead em cálculos de dimensões**
✨ **Melhor organização interna do código**
✨ **Compatibilidade ampliada com diferentes resoluções e densidades**

Esta versão representa uma **evolução significativa da biblioteca**, focada em **performance, simplicidade e flexibilidade** para projetos Android modernos.
